# Project Rules

这是一个 Unity 单机修仙剧情游戏。

## 游戏方向
- 单机游戏
- 不做联网
- 不做 PVP
- 不做排行榜
- 不做商城
- 核心玩法是格子地图探索、剧情选择、NPC 对话、每日行动点、修炼养成

## 当前玩法规则
- 地图由格子组成，每个格子有 x,y 坐标
- 玩家只能点击当前位置上下左右相邻格子移动
- 玩家不能一次移动多格
- 移动不消耗行动点
- 对话不消耗行动点
- 购买不消耗行动点
- 每天有 3 点行动点
- 修炼、采药、搜索等地点行为消耗行动点
- 行动点用完后，可以移动和对话，但不能执行消耗行动点的行为
- 点击“结束今日”后，天数 +1，行动点恢复到 3

## 技术规则
- 使用 Unity C#
- 使用 UGUI
- 数据尽量用 JSON
- 存档用本地 JSON
- 代码要简单，适合新手
- 重要脚本要写中文注释
- 不要引入联网包
- 不要引入付费插件
- 不要修改 Library、Temp、Obj 文件夹

## 文件夹规则
脚本放到：
Assets/_Game/Scripts

数据放到：
Assets/_Game/Resources/Data

场景放到：
Assets/_Game/Scenes

预制体放到：
Assets/_Game/Prefabs

## 每次完成后
- 告诉我改了哪些文件
- 告诉我怎么在 Unity 里测试
- 不要一次性实现太多系统
